Notifications - 19-178

Same Day International Phase 1: Implementation on 29 September
19-178, 23 September 2019

On 29 September 2019, ATPCO is implementing Same Day International, which will allow you to file international fare, rule, footnote, and routing changes that take effect on the same day, as well as remove or adjust unintended international fares or rule provisions from the marketplace with immediate effect.   

How to use Same Day International

The date will default to the next day, which is the current behavior as previously decided by the industry (airlines).  Watch this step-by-step training video to learn how to use this solution to save time and code efficiently. The video explains how to achieve same-day effectiveness whether you code data in our online applications or use our automated upload tools.

Other changes for implementation

The current solution for the suppression of sales for International fares will be disabled on 29 September 2019, as displayed below. The option will be enabled once again with 15-minute solution for the suppression of sales for international fares implementation planned for June 2020.


ATPCO will implement the solution for the suppression of sales for International fares on June 2020 to make it possible for you to quickly remove an unintended fare or Fare by Rule Index from the international marketplace.

Prepare for implementation

Review the video and don’t forget to familiarize yourself with the new functionality in the Gold Release testing environment.
